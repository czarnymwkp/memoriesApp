Description:
Memories is a modern web application designed for professional photographers and their clients. It allows users to securely log in and view their private photo and video sessions in a clean, Netflix-style interface. Each client receives individual login credentials to access their personalized gallery with photos, videos, and sound effects for a fully immersive experience.

Tech Stack:

Frontend: React + TypeScript

Styling: styled-components

Routing: React Router

Backend & Auth: Firebase (Authentication, Firestore, Storage)

Key Features:

Secure client login system

Responsive photo and video gallery

Custom audio effects per album

Photographer dashboard for managing clients and sessions

Cloud-based storage and delivery via Firebase

Goal:
To provide photographers with an elegant, subscription-based platform to deliver professional photo and video sessions directly to clients.
